Gas Pressure <-> Temperature conversion tester Script

 

This script will allow you to test the gas conversion of PressToTemp and TempToPress Function Block

The files can be found in https://svncarel.carel.com/svn/csuite_help/trunk/c.library_ENG/script/test_gas_conversion

The ConversionTester application is needed (search it in storage)

Prerequisites
  • Python 2.7.x freely available from https://www.python.org/downloads/, remember to flag the add to PATH option during the installation customization
  • pymodbus python library: open windows cmd and paste:

    pip install --proxy=https://haproxy.carel.locale:8080 pymodbus

 

Settings
  • Upload to a controller the ConversionTester application to a controller
  • Open the script with a text editor and change the ip variable at line 9 with the one of your controller
  • Execute run_test.bat

the command line will show the result of the tests, if any error occurs you can check inside the "result" folder, each block has its own log for every gas, some log examples are provided in the folder

 

How to add a new gas
  • Export the gas properties (pressure <-> temperature) from refprop and put the .csv in the gas folder
  • Open the script and locate the TempToPress_REAL class
  • Copy a "test_*" method and rename it with the new gas name
  • Edit gas_number and gas_name accordingly
  • locate the PressToTemp_REAL class

  • Copy a "test_*" method and rename it with the new gas name

  • Edit gas_number and gas_name accordingly (pay attention if the new gas has a bubble point)
