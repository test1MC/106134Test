Flag Table Generator Script

 

This script will allow you to create a table that contains all flag from a variable export of a POU, the generated table can be copied in flare in Text Editor mode.

The files can be found in https://svncarel.carel.com/svn/csuite_help/trunk/c.library_ENG/script/datatypeTableCreator

Prerequisites
  • Python 2.7.x freely available from https://www.python.org/downloads/, remember to flag the add to PATH option during the installation customization
  • Dominate python library https://github.com/Knio/dominate for more information on how to add a library go to https://docs.python.org/2/install/ and https://pypi.python.org/pypi, or more simply open windows cmd and paste:

    pip install --proxy=https://haproxy.carel.locale:8080 dominate
  • Classes python library

 

Settings

Open the script with a text editor and change the path of:

  • projectFile variable to your project isaxml (i.e. Desktop\cpCOe_2\cpCOe_2\cpCOe_2\cpCOe_2.isaxml), use the one from the library solution not the demo
  • destinationFile variable to a desired path

Once these variables are properly set simply launch the script, you will see "finished" once the script has ended.
