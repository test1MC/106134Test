Sniffer for RS485

Guide to setup of IF TOOLS - analyser serial RS485 IF-MSB-RS485

 

     

Analyser RS485 serial IF-MSB-485, is a sniffer to intercept, store and analyze network traffic in a serial 485 (in this case Modbus RTU)

 

1.Download from this link and install "msb-4.2.0-windows-installer.exe" (or latest version)
2.Start from icon link
3.Select "Offline MSB-485" if you need to open a log generated previously, otherwise if the sniffer is connected you can skip this step
4.Select comunication parameters
5.Press key on "Protocol monitor"
6.Select Protolcol template "Modbus RTU"
7.Press key "Start recording"
8.Working
9.Press "Lock current view" to freeze the traffic
10.Press "Autoscrolling current view" to see all traffic
11.Press "Synchronize current view with other client" if you have more tools opened at the same time and you want to show the same thing in both windows
I.E. protocol monitor and data monitor
