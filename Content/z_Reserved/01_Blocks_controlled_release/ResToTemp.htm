General operations

ResToTemp converts the value from a resistance probe into a temperature value by applying the Steinhart-Hart equation, which is the following:

 

1 T =   A ⁢ +   B ⁢   ln ⁡ ( R ) + C ⁢ ( ln ⁡ ( R ) ) 3

T is the temperature in Kelvins.

R is the resistance at T in ohms.

A, B and C are the Steinhart-Hart coefficients.

Steinhart-Hart coefficients

The formula requires three coefficients (A, B and C) to calculate the S-H model.

A, B and C depends on the model of thermistor and the temperature range of interest.

These coefficients are usually given on probe technical leaflets and may vary depending on the ideal range of work of the probe.

S-H Coefficients can be calculated from 3 given theoretical points of resistance and its corresponding temperature with a more complex formula. Many technical websites provide tools to calculate them.

To find S-H coefficients we need at least three operating theoretical points, which may be given on probe specifications.

The following equations are used to calculate the coefficients when these are not available from technical specs.

L 1 = ln ⁡ ( R 1 ) ,   L 2 = ln ⁡ ( R 2 ) ⁢ ,   L 3 = ln ⁡ ( R 3 )

Y 1 = 1 T 1 , Y 2 = 1 T 2 , Y 3 = 1 T 3

γ 2 ⁢ =   Y 2 − Y 1 L 2 ⁢ − L 1 , γ 3 ⁢ =   Y 3 − Y 1 L 3 ⁢ − L 1

⇒C=(γ3− γ2L3⁢− L2)(L1+L2+L3)−1

⇒B=γ2− C(L12+L1L2+L22)

A ⁢ =   Y 1 ⁢ −   ( B ⁢ +   L 1 2 C )   L 1

For best results, two of the three R-T points to be chosen must be close to the extremes of the operating range, and the third should be on the middle of it.

Inputs

Block pin list

Variable Data types Description
ResPrbVal REAL Resistance value from probe (ohm)
A_Coeff REAL Coefficient "A" of S-H equation
B_Coeff REAL Coefficient "B" of S-H equation
C_Coeff REAL Coefficient "C" of S-H equation
SenTyp BOOL Sensor type (0:14; 1:15)
En_ResToTemp BOOL Enable function block

 

SenTyp input specifies if, in c.design, is used type 14 (Reserved 1) or type 15 (Reserved 2). Type 14 is for resistance probe which range goes from 0 to 300kOhm, while type 15 is from probes with range from 0 to 600kOhm. The calculation needs to multiply the value of the probe by 5 if type 14 or by 10 if type 15.
