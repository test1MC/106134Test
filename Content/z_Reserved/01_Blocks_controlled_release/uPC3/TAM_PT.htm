General operations

TAM_PT function block converts the value read by the TAM and PT input of the controller to usable values.

TAM_PT also calculates the power values and the total energy consumption.

 

The calculations done by this block are valid only in a sinusoidal system.

You can initialize the block by setting a value to EnergyConsumptRetainIn before enabling the block.
