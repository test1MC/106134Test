Project description

With this demo it is possible to:

  1. manage Copeland Scroll™ Digital Compressor compressors
    1. install defaults:

      1. Envelope default;

      2. Compressor default;
    2. filter evaporation temperature;
    3. calculate envelope limits, working zone and trigger specific alarm conditions when working point is out of envelope;
    4. manage alarm delay times;
    5. manage compressor capacity and prevent actions, fan set point, electronic expansion valve MOP and cooling capacity value;
    6. manage safety times
  2. manage EVD_Emb for the superheat control

 

The following image describes the flow diagram of the software application necessary to manage a Copeland Scroll™ Digital Compressor compressor (this is similar to a step or stepless compressor, simply substitute the defaults and compressor function block).

For information on how Copeland Scroll™ Digital Compressor works, see the corresponding documentation

 

Main default settings

The following steps are needed to use the demo:

 

Main settings
Step 1 Install EVD_Emb defaults
Step 2

Select the correct model of compressor to be used
Step 3 Check the input probe values: the working point must be inside the envelope
Step 4 Set the desired capacity request from the main screen

 

c.pCOmini inputs
  • U1 : S1_SuctP suction pressure
  • U2 : S2_SuctT suction temperature
  • U3 : S3_DscgP discharge pressure
  • U4 : S4_DscgT discharge pressure

 

c.pCOmini output
  • NO1 : compressor relay
  • NO2 : cyclical relay for controlling the compressor valve

 

If using a Scroll Digital 24V valve this can be connected it to the SSR relay. If using a 220V valve, an external SSR relay is required
