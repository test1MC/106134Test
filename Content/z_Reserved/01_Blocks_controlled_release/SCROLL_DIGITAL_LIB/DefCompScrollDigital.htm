General operations

This function block provides all the necessary configuration data for Scroll Digital Compressor management, according to the compressor manufacturer and compressor model.

Inputs
Block pin list
Variable Data types UM Description

CfgScrollDigitalIn_SCRDIG

CFG_COMP_DIGITALSCROLL

 - Scroll Digital Compressor configuration
CompManuf_SCRDIG UINT -

Compressor manufacturers:

0: CUSTOM

1: BITZER

2: REFCOMP

3: FRASCOLD

4: HANBELL

5: FUSHENG

6: J&E HALL

7: COPELAND

8: DANFOSS
CompModel_SCRDIG UINT - "Compressors models supported"

En_DefCompScrollDigital

 BOOL - Enable this function block

if choosing a compressor other than Copeland Scroll™ Digital Compressor, the default will not be managed and the output will set to zero

Outputs
Block pin list
VariableVariable Data types UM Description
CfgScrollDigital_SCRDIG

CFG_COMP_DIGITALSCROLL

 - Scroll Digital Compressor configuration
En_Comp_SCRDIG BOOL - Enable compressor control
DefDone_SCRDIG BOOL - Compressor default values loaded
CurrManuf_SCRDIG

UINT

 - Current compressor manufacturer
CurrModel_SCRDIG UINT - Current compressor model
