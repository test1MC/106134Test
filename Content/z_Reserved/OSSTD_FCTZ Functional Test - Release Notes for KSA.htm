• VersionChk_2_1.0.0
• CheckRetainMem_1.2.0		 This version is compatible only with OS >= 4.x.x 

Fixed
• Fixed Digital Input class in c.mask. Before, the value "0" was "Open contact" and "1" was "Close", now it has been correct in "0" = "Close" and "1" = "Open".
• Fixed hour change in "Settings" menu. Before, it was not possible to change Minutes and Hours, due to a class assignment bug; now it has been fixed.
• Minor fixes in alarm management.

Useful information
  • ap1 files of c.pCO Functional Test don't contain OS.

  • For c.pCO, use the "Large" .ap1 file for small/medium/large models, and "Xlarge" only for Extra Large model
