Project description

With this demo it is possible to:

  1. Manage the CPY production request via Modbus Master.
  2. Communicate the CPY via Modbus Master on fieldbus2 port.
  3. Test all the device's function.
  4. Monitor all the CPY status on pGD1.
Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Set the address and enable the Modbus Master master communication.
Step 2 Change the model according to the product code of the CPY.
Step 3 Set the operation mode
Step 4 Enable the serial request and set the production request
Step 5 Start the operation on CPY
Communication settings
Serial port Fieldbus2
Protocol Modbus Master
Baudrate 9600
Data length 8
Stop bits 1
Parity None
