CPY

The CPY function block is used to manage serial communication drivers between the C.pCO and the CPY board for the KUE humidifier kit.

Block no longer in use and replaced by module CPY_2.

General operations

For a full list of board variables available for the network, see chapter Board control via network, in the board user manual.

For easier board and function block variable identification, the prefix “Ax_By” was added to each identifier where:

  • “Axx”: indicates the type of data. (A = analog, I = integer, D = digital) and the index (xx) of the supervised variable (CAREL protocol, not ready in this demo)
  • “By”: indicates the corresponding parameter number in CPY

Example: I05_B1_OptionalFunctions is the address 05 integer variable that corresponds to CPY parameter B1.

Important variables

DEFAULT INSTALLATION

Default parameters must be installed as the demo application for correct function block operations. Furthermore, the POU containing the function block and relevant default installation can be imported.

SERIAL COMMUNICATION SETTINGS

Following are the standard CPY configuration settings for serial communications with C. pCO

  • C3 – serial address (depends on the code, usually 1 or 2)
  • C4 – baud rate (0 = 9600; 1 = 19200)
  • C5 – supervisor: frame (bit per character, parity stop bit; according to programming)

To open these settings from CPY terminal, press and hold down “ENTER” in the main mask for two seconds, enter “77” and scroll masks to view the required settings. Reboot CPY control to save settings.

Changes to serial connection settings only apply after the CPY is rebooted.

OptionalFct

This is a input pin contains each bit of I05_B1_OptionalFunctions using a structure CPY_WORD_BITS.

CtrlMan

This is a input pin for the manual control to Drain and Pre-cleaning. And decide the signal from logic or mask.
