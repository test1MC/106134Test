Project description

Project aims are:

1. Showing how EVD_EVO can be managed inside a c.suite project.

2. Testing the main device functions

With this demo it is possible to:

  1. Probe readings management

  2. Example of use of alarms

  3. Default installation logic by DEV editor and c.strategy part

  4. Common functionalities are grouped into the same strategy page in order to allow a better focus and to prepare that page for the export.
