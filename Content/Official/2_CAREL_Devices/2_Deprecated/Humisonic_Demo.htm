Project description

 

Demo/Test application for the Humisonic Humidifer management using the Humisonic function block.

  • Management of the communication: Modbus Master and Carel Master on the fieldbus port.
  • Management of the Humisonic humidifier via serial communication.
Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Enable the Humisonic function block in menu CONFIG
Step 2 Set a production request in menu IN_OUT
Communication settings
Device address 1
Serial port Fieldbus
Protocol default Modbus Master, Carel Master
Baudrate 4800, 9600, default 19200, 38400
Data length 8
Stop bits 2
Parity None
Notes

The Humisonic default address is “1”. It can be changed using the device user interface (C3 parameter) or from the Generic R/W mask (“Manual Mng” loop), assigning the proper value to the I47 / HR175 parameter.
