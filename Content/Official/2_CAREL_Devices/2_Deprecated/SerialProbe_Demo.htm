Project description

With this demo it is possible to:

  1. Communicate via Modbus Master protocol with a serial probe
  2. Accesses all status variables and parameters in serial probe devices
Main default settings
Default probe network address
Address 128 (it's possible to change it by the main mask)
Communication settings
Serial port BMS2
Protocol Modbus Master
Baudrate 19200
Data length 8
Stopbit 1
Parity None
