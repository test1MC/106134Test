Project description

Project aims are:

  1. Showing how EVD_Emb can be managed inside a c.suite project.
  2. Test all the device’s functions

Demo/Test application for EVD_Emb:

  • Management of measure system conversions
  • Probe readings management
  • Example of use of alarms
  • Default installation logic by DEV editor
  • Common functionalities are grouped into the same strategy page in order to allow a better focus and to prepare that page for the export.
