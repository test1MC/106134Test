General operations

EVD_EmbComm manages shared parameters between two EVD_Emb related to different EEV. If disabled, EVD_EmbComm copies DataSlave into Data in the output pin as is. If enabled, EVD_EmbComm copies some parameters from DataMaster to DataSlave. It’s possible to chose if copy or not some parameters trought the structured variable CfgCommParams: if the son parameter is set to TRUE then the related variable is copied from Master to Slave.
