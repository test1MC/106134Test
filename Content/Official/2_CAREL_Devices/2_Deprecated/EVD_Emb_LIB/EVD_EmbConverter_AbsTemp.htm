General operations

If you need to convert a differential of temperature, use EVD_Emb_Converter_RelTemp instead.

Important: DataIn is the data structure that comes from EVD_Evo. The variables to shown in user interface are a copy of data structure. These are defined as local variables of EVD_EmbConverter_AbsTemp and flagged as “user interface” in “Public Variables Selector”.

BaseUnit and DisplayUnit inputs are used to specify the unit of measurement used (FALSE: °C, TRUE: °F).

When DataIn or local variables used in masks change, the other one is updated based on it. If both changes, base wins and display is updated.

At start-up, local variables used in masks are immediately updated base on BaseTemp.

The block is able to recognize a change of display unit at run-time and react immediately with the right conversion.
