Project description

The aim of this project is:

  1. Show how the KEC can be integrated in a c.suite project.
  2. Use of the device alarm
  3. Give the possibility to integrate quickly the device exporting and importing the related POUs: all the devices are entered
  4. Quickly test the functioning and the behaviour of the KEC device without having an application available that manages it.

 

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings
KEC Model Set it according to the pumping unit connected to the device
Unit mode Set it accordingly to the system configuration
Steps number Set it accordingly to the distribution system used
IEC regulation mode Set the control mode for the distribution system associated with the IEC rack

Enable KEC

 Set it to "YES" to enable KEC control

 

Communication settings
Device address 1
Serial port Fieldbus 2(c.pco), Fieldbus (c.pco mini)
Protocol Modbus Master
Baudrate 1200, 2400, 4800,9600, default 19200, 38400
Data length 8
Stop bits default 0=1, 1=2
Parity default None, Odd, Even
