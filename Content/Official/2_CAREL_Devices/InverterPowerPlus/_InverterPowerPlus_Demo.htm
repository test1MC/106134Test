Project description

This demo can be used to:

  1. Manage the operation of the compressor without the BLDC controls .
  2. Example of using alarms
  3. Start a project to implement with other functions

Common functions are grouped on the same strategy page in order to allow a better focus and prepare that page for exporting.

 

C.pCO I/O configuration

Digital output

Channel Description
NO1 Enable PWRP function block

 

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings

Step 1:

Enable the devices. This parameter is available in the main menu under "Settings"

Step 2:

Select the correct models of compressor to be used. This parameter is available in the main menu under the item Power +

Step 3:

Write the default parameters.

Step 4:

Switch on the unit.

Step 5:

Set power required on the main screen.

 

The PowerPlus controller can be started when the demand for power exceeds the minimum percentage of inverter startup speed for the selected compressor.

The power required for the compressor starts to exceed the minimum power required to start the compressor specified on the screen.

The PowerPlus controller can be stopped in three ways: "Force stop command" on the inverter control screen; "Shutdown" on unit On/Off; "Passage through 0 of the power required" on the inverter control screen.

 

Step 1 Select the correct model of compressor to be used
Step 2 Check the input probe values: the working point must be inside the envelope
Step 3 Set the power required on the main screen

 

Communication settings

Power+:

Device address 1
Serial port Fieldbus2
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits 2
Parity None

 

Alarm semiautomatic reset configuration

The following steps are necessary to configure the semiautomatic alarm reset:

Step 1:

Configure the number of retries and the maximum retry time. These parameters are available in the main menu under the item Power +:

Step 2:

Configure the alarms with semiautomatic reset . This parameter is available in the main menu under the item Power + :

 

Use the key at the bottom of the screen for the manual or semiautomatic configuration .
