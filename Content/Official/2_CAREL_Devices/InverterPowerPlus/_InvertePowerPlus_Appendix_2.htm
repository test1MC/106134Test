Appendix 2 : Power+ driver replacement

How to replace a Power+ driver .

 

Native revision 15/11/2016

Introduction

The InverterPowerPlus function block allows automatic reconfiguration if replacing the Power+ inverter. The new device is recognised automatically by acquiring the serial code.

 

Procedure

Proceed as follows for automatic reconfiguration when replacing the Power+:

  1. Make sure the parameter download procedure is enabled in the inverterPowerPlus function block configuration menu.

  2. Power off the c.pCO and the Power+.

  3. Replace the Power+ inverter.

  4. Power on the pCO and the Power+..

As soon as Modbus communication is established between the devices, the c.pCO checks the new serial number and compatibility of the new Power+ with the compressor used; if these two conditions are both ok, the procedure for setting the Power+ parameters to the default values for the selected compressor is started automatically.

Any changes made to the Power+ parameter settings and not saved using the procedure described in “Custom parameter management” will be lost when replacing the Power+.
