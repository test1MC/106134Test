How to implement a stable or beta library

 

Native revision 15/11/2016

INTRODUCTION

The Comp_BLDC and InverterPowerPlus function block have the same versions.
The Comp_BLDC contains the InverterPowerPlus function block. For this reason, the sequence to be respect to increase the version is as follows:

  1. InverterPowerPlus function block
  2. Comp_BLDC function block

InverterPowerPlus LIBRARY

  1. Open the InverterPowerPlus project.

  2. Increase the version in solution info. If the version is a Beta, add the letter B after the version. Use the right mouse button on Edit Solution Info to find the field to be modified.

  3. Increase the version in the Comments of InverterPowerPlus POU. If a beta version, add the letter B after the version. Enter the modification in the comment field.

  4. Open the InverterPowerPlus function block. The password is required.

  5. Increase the version of InverterPowerPlus POU.

  6. If entering a new compressor:

    • enter the electrical parameters in the LoadDefaultPSD1_PWRP POU.
  7. Build the solution.

  8. Export the library.

  9. Show block library and Store as Carel Library

  10. Follow the tutorial: How to create a template library

Comp_BLDC LIBRARY

  1. Open the comp_BLDC project.

  2. Increase the version in solution info. If the version is a Beta, add the letter B after the version. Use the right mouse button on Edit Solution Info to find the field to be modified.

  3. Increase the version in the Comments of comp_BLDC POU. If a beta version, add the letter B after the version. Enter the modification in the comment field.

  4. Open the comp_BLDC function block. The password is required.

  5. Increase the version of InverterPowerPlus POU.

  6. Import the InverterPowerPlus library. The InverterPowerPlus library must have the same version as the CompBLDC library.

  7. If entering a new compressor:

    • enter the thermodynamic parameters in the CompDef_BLDC POU.
  8. Build the solution.

  9. Export the library.

  10. Show block library and Store as Carel Library

  11. Follow the tutorial: How to create a template library
