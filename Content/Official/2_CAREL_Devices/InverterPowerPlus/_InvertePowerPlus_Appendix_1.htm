Appendix 1 : custom parameter management

How to create a custom compressor.

 

Native revision 15/11/2016

Introduction

The InverterPowerPlus function block allows a compressor’s electrical parameters o be modified and saved to the c.pCO memory, thus creating a custom compressor.

 

Procedure

The procedure for creating a custom compressor is as follows:

  1. From the list in the InverterPowerPlus function block, select the compressor with the features that most closely resembles the custom compressor being created.

  2. Start the procedure to download the default parameters. This operation sets a value for all the Power+ parameters.

  3. Modify the compressor’s electrical parameters. The modified parameters are not yet saved to the c.pCO memory.

  4. Save the new compressor configuration in the Custom configuration menu.

  5. If entering a new compressor:

    • in the main_init task , Sel_Comp_Type module, create the variable with the new name of the compressor and its allocation number.
    • in the main task , createa new folder with the new compressor parameters. Keep the sequence of the folder.

After having saved the configuration as described in point 4, the list of compressors in the InverterPowerPlus function block will now show the new configuration under the CUSTOM model.
