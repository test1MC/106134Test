General operations

PSALB FB manages communication with the PSALB card, designed to be connected to the DC bus of a Carel Power+ drive and to operate as an interface for fans or pumps equipped with integrated speed control powered in DC and controlled by a 0/10 Volt signal.

It is possible to manage two loads independently. Each one is controlled by its own 0/10V signal and is equipped with an input line for speed signal.

The serial line protocol is Modbus® RS485.

For further information please refer to the product manual.

Figure 1

The PSALB CARD. (Picture taken from Power+ documentation.)

Load management

Once configured, the PSALB FB can manage the PSALB card to start/stop and/or set the speed for both loads independently. For further detail on management of the loads, please refer to the card manual.

Modbus® commands

The PSALB card uses only Registers (16 bit), not Boolean variables (coils). The implemented Modbus® functions are the following:

Function number Function name
03 (0x03) Holding register read
04 (0x04) input register read
06 (0x06) single register write

The Modbus® exceptions allowed are:

Exception number description
1 function not supported
2 address not allowed
4 value not allowed
6 device busy

Communication settings

The serial line configuration parameters are stored in the permanent memory of the card. On activation of the card, their values are set to the latest stored value.

Micro-switch 1 manages the configuration of the serial line:

Micro-switch 1 Serial line setting
OFF Configuration as parameters
ON default: 19200 bit/s; no parity; 2 stop bits; address 3

The configuration parameters are saved in the permanent memory.
When turning the card on, if micro-switch 1 is set to "ON", the serial line is configured according the default values, but the respective parameters are not modified: their values are fixed to the previous stored ones.


Before approaching to change the configuration parameters, be sure micro-switch 1 is set to "ON".
Otherwise any changes to serial line configuration parameters will not be effective.
Changes to serial line configuration parameters are allowed only if both loads are disabled.
The change of the baudrate, dataframe and card address becomes effective only after a restart or after a reset alarm command.
The configuration parameters are effectively written to the permanent memory within 10 secs since the Modbus write command. Don't switch off the power or enable the loads within these 10 secs.

For further information please refer the card manual.
