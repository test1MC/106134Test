Project description

PSALB card for auxiliary loads of 400Vdc is designed to be connected to DCbus of a drive Carel Power+ and to operate as an interface toward fans or pumps provided with a speed control integrated, supplied in DC voltage and driven by a signal between 0-10V.

Two loads can be controlled independently. Each load is driven by its dedicated signal 0-10V and is provided by an input for a tachometric signal.

The communication is performed on serial line RS485 with modbus protocol.

By this Demo you can configure PSALB card and send commands via serial line to start, stop and set speed of loads. Modalities of operations are listed below.

Load enable / disable
Load Description
enabled the command signal Vsp is generated according to relative parameters
disabled the command signal Vsp is forced to 0V (10V if inverse logic is selected)
Command logic
Logic Description
direct the command signal Vsp is directly proportional to "speed reference" parameter, that is given as integer 0-1000 corresponding to 0,0 - 100,0% Volt.
inverse the command signal Vsp is inversely proportional to "speed reference" parameter, that is given as integer 0-1000 corresponding to 0,0 - 100,0% Volt.
Loads management in case of serial line off-line.

If PSALB card does not receive data for a time longer than the one set as "off-line alarm", loads are managed as follows

Action Description
no action command signal Vsp is managed normally as set (card does no action)
disabled load

command signal Vsp is forced as disabled (see load enable/disable table above)

 

For any further detail about the real command signal Vsp, alarms and Modbus command, complete table of parameters please refer to data-sheet card inside Power+ manual.

Wizard for serial line parameters setting

Parameters related to serial line communication are stored in permanent memory. At starting, their value refreshes to latest set. If you have to modify the serial line configuration of PSALB, pay attention to follow the correct procedure mask by mask in the specific wizard:

  • Card writes communication parameters in permanent memory only if both loads are disabled. The first mask warns user about this.
  • Fill fields with new values.
    Setting "YES" in "Modify" field and pressing ENTER, card writes and changes the communication parameters.
    Please wait and do not enable loads or remove power supply until the message "Operation completed!" appears on screen.
