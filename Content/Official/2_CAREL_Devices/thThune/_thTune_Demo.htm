Project description

With this demo it is possible to:

  1. Manage one th-Tune room terminal in all its functions and features
  2. Communicate with one th-Tune room terminal using Modbus Master protocol
Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Enable the th-Tune function block in the MAIN menu
Step 2 Enter the TH-TUNE menu and select the functions to enable/disable
Communication settings
Device address 1 (editable)
Serial port Fieldbus
Protocol Modbus Master
Baudrate 4800, 9600, default 19200
Data length 8
Stop bits 2
Parity None
Unit of Measure management

The unit of measure can be changed:

  1. By input pin "UnitMeasure_THTN": this pin sets the protocol and thThune UoM
  2. On the Main screen: this function sets the UoM only for screen display
Notes

A set of default values is automatically installed after initialisation. In this case, the device address is 1.
