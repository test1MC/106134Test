Project description

This demo can be used to:

  1. Manage a Wireless probe (SA or SI) to read temperature, humidity and light intensity from the field

  2. Communicate with a Wireless Access Point using Modbus Master protocol

 

Main default settings

 

Main settings
Step 1 Bind the Wireless Probe to the Access Point
Step 2 Set the probe address in the main screen and enable the Function Block
Step 3 Change the settings of the probe from the WIRELESS PRB CFG menu

 

Communication settings
Device address 16
Serial port Fieldbus2.
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits 2
Parity None
