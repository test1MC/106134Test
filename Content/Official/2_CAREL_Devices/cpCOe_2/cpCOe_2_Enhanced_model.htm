Useful information

To manage the c.pCOe Enhanced model, the"cpCOe_2 "FB is used . There are no parameters to set that distinguish the Basic model from the Enhanced model, it is the FB that recognises which device is connected.

This new model can work in a Modbus network and can replace a Basic model with the same functions and, in addition, the connection of an unipolar electronic expansion valve.

 

c.pCOe Enhanced devices works in Modbus slave protocol, not in Carel Slave protocol. Also, the supported Baudrates are 19.2k, 38.4k and 57.6k.

c.pCOe Enhanced devices cannot manage the valve, it only sends information to the valve driver.

Valve management

c.pCOe Enhanced model manages only unipolar electronic expansion valves, such as Carel E2Vu, according the"EVD_Emb_2"FB supported valves. Through ExtPosit pins, the information is shared between cpCOe_2 and EVD_Emb_2 function blocks.

Polling Time

The output pin ThrshWarnPollTime_CPCOE of "cpCOe_2 "FB warns that the Polling Time exceeds 10 seconds.

This function works only with the Enhanced model because a polling time greater than 10 seconds makes control performed by the valve inaccurate.
