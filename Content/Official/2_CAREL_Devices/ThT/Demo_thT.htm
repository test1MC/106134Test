Project description

This demo can be used to:

  1. Manage a thT room terminal in all its functions and features

  2. Communicate with a thT room terminal using Modbus Master protocol

 

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings
Step 1 Enable the thT function block in the MAIN menu
Step 2 Enter the thT CFG menu and configure the terminal functions

 

Communication settings
Device address 50
Serial port Fieldbus2.
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits default 0=1
Parity None
