General operations

The thT function block is used to communicate with the thT device.

With this function block you can:

  • manage the relay manually
  • enable and disable the thT
  • enable and disable the keylock
  • change the operating mode of the thT
  • change the fan and heat level separately
  • change the clock
  • read temperature and humidity if sensor is present (no alarms will be activated if sensor is missing)
  • edit time bands

 

Configuration structure

With this structure you can:

  • Manage the relay manually, set RelayMng to TRUE then edit RelayManOnOff
  • Enable and disable the thT, set OnOffSelect to TRUE then edit OnOffCtrl, OnOffSelect can then be set to FALSE if you want to change the status of the thT by the terminal
  • Enable and disable the keylock, edit KeyLockFuncEn
  • Enable and disable the buzzer, edit BuzzEn
  • Change the fan and heat level, set FanHeatBtnSelect to TRUE, edit FanBtnSet and HeatingBtnSet, use FanHeatBtn to choose between the two values
  • Change the clock, edit Hour, Minute and Day
  • Change the set points, edit TempSetP, EcoSetP and PartySetP
  • Change the offset for the probes, edit TempOffset and HumOffset

  • Change the clock button behaviour, set ClockBtn to the desired value, this is used to hide some possibilities when pressing the clock button

    so the value of ClockBtn must range from 0 to 7

  • Change the working mode of the thT, set ModeBtnSelect to TRUE, edit ModeBtn and ModeBtnValue

    so the value of ModeBtn must range from 0 to 511, the actual mode can be edited in ModeBtnValue, the values of this variables range from 0 to 8 according to the bit number that enables the mode

  • Enable frost mode, set FrostModeEn to TRUE and edit FrostTemp

On/Off, Mode and Fan/Heating button can be edited exclusively from the device or from the controller, each have a selector in the configuration structure.

Set the values in this structure correctly otherwise communication may stop (i.e. Day = 0).

 

Time band structure

With this structure you can enable and modify the time bands on the thT, every band is grouped inyo a structure

The bands are divided into Monday-Friday and Saturday-Sunday with 4 bands for each group

Each band contains the start time of the band and the set point to use during that band
