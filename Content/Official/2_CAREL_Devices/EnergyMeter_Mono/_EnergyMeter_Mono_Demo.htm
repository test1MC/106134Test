Project description

Project aims:

  1. Show how an emeter1 can be managed inside a c.suite project

  2. Test all the device’s functions

With this demo it is possible to:

  1. Communicate via Modbus Master protocol with a single-phase energy meter (emeter1)

  2. Access all status variables and parameters on emeter1 devices
