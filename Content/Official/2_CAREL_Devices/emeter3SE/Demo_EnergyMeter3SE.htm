Protocols Modbus Master

 

Project description

This demo can be used to:

  1. Communicate with an Energy Meter 3 SE

  2. Access all status variables and parameters on Energy Meter 3 SE devices

 

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings
Step 1 Set the address and enable the EnergyMeter3SE function block in the MAIN menu
Step 2 Enter the ENERGY METER menu to see all values of the device

 

Communication settings
Device address 1
Serial port Fieldbus2
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits 1
Parity None
