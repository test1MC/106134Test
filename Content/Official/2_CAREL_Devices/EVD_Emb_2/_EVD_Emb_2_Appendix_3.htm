Appendix 3: benchmark comparison between Mod_EVDEVO and EVD_Emb_2

Last update 06/05/2015 | Native revision - 20/01/16

 

 

Here below is an overview of the control performance on a c.pCOmini using the EVD_Emb_2 function block. The reference used for comparison is the EVD Evolution parametric controller. Using the same application, the same stimulus was applied to the system; the response was recorded in terms of valve position and thermodynamic key components of superheat control. This kind of approach was followed, changing the length of the program cycle time (in ms) and the time constant for digital probe filtering (K).

As can be seen in the graphs, the dynamics and the working point reached by EVD_Emb_2 function block is the same as EVD Evolution.
