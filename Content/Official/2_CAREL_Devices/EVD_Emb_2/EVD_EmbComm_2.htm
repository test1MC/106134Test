General operations

EVD_EmbComm_2 manages shared parameters between two EVD_Emb_2 relating to different EEVs. If disabled, EVD_EmbComm_2 copies ParamsSlave to Params_EVDEMB on the output pin as is. If enabled, EVD_EmbComm_2 copies some parameters from ParamsMaster to ParamsSlave. It is possible to choose whether or not to copy some parameters using the structure CfgCommParams: if the parameter is set to TRUE, then the related variable is copied from Master to Slave.
