Project description

With this demo it is possible to:

  1. Communicate via Modbus Master protocol with a serial probe
  2. Set the Unit of Measure (screen)
  3. Access all status variables and parameters on serial probe devices
Main default settings
Default probe network address
Address 128 (this can be changed on the main screen)
Communication settings
Serial port FB2
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits 1
Parity None
