Project description

With this demo it is possible to:

  1. Manage the CPY production request via Modbus Master.
  2. Communicate with the CPY via Modbus Master.
  3. Test all the device's functions.
  4. Monitor the CPY status on pGD1.
Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Set the address and enable Modbus Master master communication.
Step 2 Change the model according to the CPY product code.
Step 3 Set the operating mode
Step 4 Enable the serial request and set the production request
Step 5 Start CPY operation
Communication settings
Device address 1
Serial port Fieldbus1
Protocol default Modbus Master, Carel Master
Baudrate 9600, default 19200
Data length 8
Stop bits default 0=1, 1=2
Parity None
