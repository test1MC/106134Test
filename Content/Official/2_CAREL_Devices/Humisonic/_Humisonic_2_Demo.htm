Project description

 

Demo/Test application for the Humisonic humidifer management using the Humisonic_2 function block.

  • Management of communication: Modbus Master and Carel Master on the fieldbus port.
  • Management of the Humisonic humidifier via serial communication.
Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Enable the Humisonic_2 function block
Step 2 Set a production request other than zero
Communication settings
Device address 1
Serial port Fieldbus
Protocol default Modbus Master, Carel Master
Baudrate 4800, 9600, default 19200, 38400
Data length 8
Stop bits 2
Parity None

 

In protocol editor, make sure the Modbus line is first (ID = 0) and then the CAREL line (ID = 1)

Notes

The Humisonic default address is “1”. This can be changed using the device user interface (C3 parameter) or from the Generic R/W screen (“Manual Mng” loop), assigning the desired value to parameter I47 / HR175.
