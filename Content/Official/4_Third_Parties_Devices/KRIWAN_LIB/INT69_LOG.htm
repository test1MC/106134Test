General operations

The INT69_LOG function block reads the INT69® protection module error data logger information via Modbus protocol.

The information shown in masks are:

  • Error log message

  • Error log time stamp since last power reset of INT69 protection module

In masks it shows:

  • The last error
  • Scroll all the memory error (19 records)

for information on the INT69 technical specifications, Modbus compatible models and all other details, visit the KRIWAN Industrie-Elektronik website.

 

Communication settings

Modbus Master

For communications via Modbus Master protocol, the INT69_LOG function block requires the template saved inside ".otlib" file:

  • INT69_LOG_MBM_MEMORY_ERROR.txt
