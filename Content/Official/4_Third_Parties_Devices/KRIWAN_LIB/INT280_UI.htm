General operations

The INT280_UI function block reads information from INT280 function block in order to manage the summary mask of this device:

Inputs

Block pin list

Variable Data types UM Description
GenStat_INT280 READ_MODULESTATUS - General module status
PwrSupplyStat_INT280 READ_MODULESTATUS - Power supply status
OilLvlLowStat_INT280 READ_MODULESTATUS - Oil level Low status
WarningsStat_INT280 READ_MODULESTATUS - Warnings status
CommStat_INT280 READ_MODULESTATUS - DP communication status
StatusMsg_Bit1_3_OilMonitoring_INT280 READ_MODULESTATUS - Oil monitoring: 0 = oil available 1 = filling active; 2 = pause active; 3 = overfilling active; 4 = oil shortage + filling active; 5 = oil shortage + Pause active
En_INT280_UI BOOL - Enable of INT280_UI function block
Outputs

Block pin list

Variable Data types UM Description

Stat_INT280_UI

 USINT - Status 1 = Oil available, level OK, 2 = filling active; 3 = overfilling active; 4 = pause active ; 5 = oil level too low; 6 = internal error
ProtectionStat_INT280_UI UINT [0..1] - Protection Status ([0] = number of active protections, [1] = which protection is active)

 

For data types meaning refer to INT280 and INT280_LOG data types list
