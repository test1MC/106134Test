General operations

The INT280_Oil_Level_Regulator function block reads the INT280® Oil level regulator, via Modbus protocol, information:

  • Device status
  • Oil valves status, switching frequency and working time 

The information shown in masks are available according to INT280® model. The function block is able to auto-recognize the connected protection module.

for information on the INT280® technical specifications, Modbus compatible models and all other details, visit the KRIWAN Industrie-Elektronik website.

Communication settings

Modbus Master

For communications via Modbus Master protocol, the INT280_Oil_Level_Regulator function block requires the templates saved inside ".otlib" file:

  • INT280_MBM_VISU_ID_CHK.txt
  • INT280_MBM_VISU_ID_1.txt
  • INT280_MBM_VISU_ID_2.txt

It is possible to force the usage of only one template, if the visualization ID of the INT280 device has been known. Setting the En_FrcVISU_ID_INT280 to TRUE the FB will enable the requested VISU_ID_x template, according to FrcVISU_ID_INT280 value.

Alarms

INT280_OilLevelRegulator function block has the following alarms. In the table below there are the possible causes and the suggested actions for software application.

Alarm Possible causes Suggested actions
Offline alarm Wrong serial address
 No action
Error alarm Another alarm has been triggered (except for offline alarm) Turn off the compressor
Use the memory error of INT280, last error variables (INT280_LOG)
General module Internal error of INT280 device No action

Note: it could be necessary change INT280 device, because it has been shutdown by an internal error
Power supply module Power supply of INT280 is not regular No action

Note: it could be necessary to check the power supply of INT280 device
Oil level low Empty oil collector
Low differential of pressure between the compressor and oil collector		 No action

Note: it could be necessary to check the oil collector or check oil differential pressure between compressor and collector
Warnings Conditions that aren't possible (ex. oil is refilling but the compressor is off) Use the memory error of INT280 (INT280_LOG) to understand the possible cause of this alarm
DP communication An error coming from the connected INT69 device No action
