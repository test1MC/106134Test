General operations

The INT69 function block reads the INT69® protection module information via Modbus protocol:

  • Protection module status

  • Working hours

  • Compressor Motor and alarm status

  • Value read by sensors (PTC probes for Motor and Discharge temperatures)

The information shown in masks are available according to INT69® model. The function block is able to auto-recognize the connected protection module.

for information on the INT69® technical specifications, Modbus compatible models and all other details, visit the KRIWAN Industrie-Elektronik website.

 

Communication settings

Modbus Master

For communications via Modbus Master protocol, the INT69 function block requires the templates saved inside ".otlib" file:

  • INT69_MBM_VISU_ID_CHK.txt
  • INT69_MBM_VISU_ID_2.txt
  • INT69_MBM_VISU_ID_4.txt
  • INT69_MBM_VISU_ID_5.txt
  • INT69_MBM_VISU_ID_6.txt
  • INT69_MBM_VISU_ID_8.txt
  • INT69_MBM_VISU_ID_9.txt

It is possible to force the usage of only one template, if the visualization ID of the INT69 device has been known. Setting the En_FrcVISU_ID_INT69 to TRUE the FB will enable the requested VISU_ID_x template, according to FrcVISU_ID_INT69 value.

Alarms

INT69® has the following alarms. In the table below there are the possible causes and the suggested actions for software application.

Alarm Possible causes Suggested actions
Offline alarm Wrong serial address
 No action
Error alarm Another alarm has been triggered (except for offline alarm) Turn off the compressor
Use the memory error of INT69, last error variables ( INT69_LOG)
Motor temperature PTC probe module Maximum motor temperature exceeded
High motor temperature at power on
Probe broken or unreliable reading
Probe short circuit
Probe open circuit
 Turn off compressor
If available, enable algorithm to decrease motor temperature (with Liquid injection/fans/etc..)

Note: it could be necessary to check the PTC probe
Motor temperature PTC probe module dynamic Motor temperature has grown unusually fast Turn off compressor
If available, enable algorithm to decrease motor temperature (with Liquid injection/fans/etc..)

Note: it could be necessary to check the PTC probe
Discharge temperature PTC probe module Maximum discharge temperature exceeded
High discharge temperature at power on
Probe short circuit
Probe open circuit		 Turn off compressor
If available, enable algorithm to decrease motor temperature (with Liquid injection/fans/etc..)

Note: it could be necessary to check the PTC probe
Relay manipulation module Compressor should be off while for INT69 device is on Turn off the compressor
On/Off compressor cycle module Compressor has been turn on and off too many times in few time Check the regulation algorithm and compressor management in application
INT250 Oil differential pressure probe status Differential oil pressure probe not installed
Differential oil pressure too low
Probe short circuit
 Turn off the compressor

Note: it could be necessary to check the PTC probe, oil level and oil pump (it can be broken)
Phase monitoring module Compressor power supply phase loss Put the compressor in "error condition" (alarm)
Turn off the compressor
General module Internal error of INT69 device No action

Note: it could be necessary change INT69 device, because it has been shutdown by an internal error
Power supply module Power supply of INT69 is not regular No action

Note: it could be necessary to check the power supply of INT69 device
