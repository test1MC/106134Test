Project description

With this demo it is possible to:

  1. Manages the communication between a c.pCO and a Ziehl-Abegg fan
  2. Run the fan according the control modes available (“absolute speed” control [rpm] and “0-100.0%” modes).

  3. Monitoring the operating status of the fan (Temperature, power supply, current, …)
