Hardware and software requirements
c.suite version ≥ 3.0.41
Hardware c.pCO mini
O.S. ≥ 3.0.001
Devices ZiehlAbegg fan
Devices firmware -
Protocols Modbus Master
Project description

With this demo it is possible to:

  1. Manage communication between a c.pCO and a Ziehl-Abegg fan
  2. Run the fan according the control modes available (“absolute speed” control [rpm] and “0-100.0%” modes).

  3. Monitoring the operating status of the fan (temperature, power supply, current, …)

Main default settings

The following steps are necessary in order to use the demo:

Main settings
Step 1 Enable function block/communication from screen (from the speed set point screen, press the Down button)
Step 2 Set a speed set point
Communication settings
Device address 247
Serial port Fieldbus
Protocol Modbus Master
Baudrate 19200
Data length 8
Stop bits 1
Parity None
