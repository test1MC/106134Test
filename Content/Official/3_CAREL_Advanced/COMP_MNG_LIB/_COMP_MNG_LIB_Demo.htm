Project description

This demo can be used to manage step and stepless compressors:

  1. install defaults:
    • Envelope defaults;
    • Compressor defaults;
    • Valve defaults;
  2. calculate envelope limits, working zone and trigger specific alarm conditions when working point is out of envelope;
  3. manage alarm delay time;
  4. manage compressor capacity and prevent actions, fan set point, electronic expansion valve MOP and cooling capacity value;
  5. manage safety times

The following image describes the flow diagram of the software application needed to manage a stepless compressor (for a step compressor, the diagram is the same with the default and compressor management function blocks relative to step instead of stepless compressor).

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings
Step 1

Select the correct model of compressor to be uses
Step 2 Check the input probe values: the working point must be inside the envelope. See typical values in the universal input table .
Step 3 Set required capacity from the main screen

If everything is OK, then the compressor will turn on (NO1 close) and the status on the main screen will be ON. More info about compressor: PRG ->COMP.SCREW -> the first 3 screens summarise the status of the compressor
