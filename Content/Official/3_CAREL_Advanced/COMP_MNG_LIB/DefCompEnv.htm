General operations

This function block provides all the necessary configuration data for envelope management according to the compressor manufacturer and compressor model.

 

Inputs

Block pin list

Variable Data type UOM Description
CfgEnvIn CFG_ENV_COMP - Envelope parameters
CompManuf UINT -

Compressor manufactures:

0: CUSTOM

1: BITZER

2: REFCOMP

3: FRASCOLD

4: HANBELL

5: FUSHENG

6: J&E HALL

7: COPELAND

8: DANFOSS
CompModel UINT -

 
En_DefCompEnv BOOL - Enable this function block

 

Outputs

Block pin list

Variable Type UOM Description
CfgEnv CFG_ENV_COMP - Envelope configuration
RfrgTyp INT - Refrigerant type
DefDone BOOL - Default values of envelope are loaded
CurrManuf UINT - Current compressor manufacturer
CurrModel UINT - Current compressor model
