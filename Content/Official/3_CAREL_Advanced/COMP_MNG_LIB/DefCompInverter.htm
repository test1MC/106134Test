General operations

This function block provides all the necessary configuration data for screw compressor management according to the compressor manufacturer and compressor model.

Name Description Data type
CFG_COMP_INVERTER Compressor inverter configuration

CFG_COMP_INVERTER
  MinPwr Minimum capacity REAL
  MaxPwr Maximum capacity REAL
  StepNo Step number UINT
  PR_ThrshStep1Step2 Pressure ratio threshold step 1-2 REAL
  PR_ThrshStep2Step3 Pressure ratio threshold step 2-3 REAL
  PR_ThrshStep3Step4 Pressure ratio threshold step 3-4 REAL
  MinMinOnT Lower limit for minimum compressor ON time UINT
  MinMinOffT Lower limit for minimum compressor OFF time

UINT
  DTStep1Step2 Delay between step 1-2 UINT
  DTStep2Step3 Delay between step 2-3 UINT
  DTStep3Step4

Delay between step 3-4

 UINT
