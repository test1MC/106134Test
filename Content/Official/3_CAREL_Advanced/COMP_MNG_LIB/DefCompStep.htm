General operations

This function block provides all the necessary configuration data for step compressor management according to the compressor manufacturer and compressor model.

 

Inputs

Block pin list

Variable Data type UOM Description

CfgCompStepIn

CFG_STEP_COMP

 - Step compressor configuration
CompManuf UINT -

Compressor manufacturer:

0: CUSTOM

1: BITZER

2: REFCOMP

3: FRASCOLD

4: HANBELL

5: FUSHENG

6: J&E HALL

7: COPELAND

8: DANFOSS
CompModel UINT -

"Compressor models supported"

En_DefCompParams BOOL - Enable this function block

 

Outputs

Block pin list

Variable Data Type UOM Description
Cfg_CompStepOut CFG_STEP_COMP -

Step compressor configuration

 
En_CompStep BOOL - Enable STEP compressor
DefDone BOOL - Default values for the compressor are loaded
CurrManuf UINT - Current compressor manufacturer
CurrModel UINT - Current compressor model
