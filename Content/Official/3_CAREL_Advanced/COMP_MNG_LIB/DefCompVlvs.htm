DefCompVlvs

Provides all the necessary configuration data for compressor valve management

General operations

This function block provides all the necessary configuration data for valve management according to the compressor manufacturer and compressor model.

The following table shows the cross-reference between valve number and valve name for different compressors.

  Bitzer Hambel Refcomp Frascold
Valve 1 CR4 - - -
Valve 2 CR2 15 15/21 V2
Valve 3 CR3 14 14/20 V1
Valve 4 CR1 16 16 V3
