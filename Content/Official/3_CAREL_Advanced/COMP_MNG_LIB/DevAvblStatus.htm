Checks if a compressor is available

General operations

Based on the DevAvblStatus input signal, a variable is created that summarises the compressor status, useful as feedback for the CompStepless, CompStep FBs. This can be used on the user interface to show the status of the compressor.

The following is the explanation and the priority:

 

  • IF DevAlrm THEN DevAvblStatus = 10
  • IF ManMode THEN DevAvblStatus = 13
  • IF FrcOffDev THEN DevAvblStatus = 6
  • IF FrcdOffByT THEN DevAvblStatus = 11
  • IF FrcdOnByT THEN DevAvblStatus = 12
  • IF PmpDwnRun THEN DevAvblStatus = 14
  • IF AdmitMaxPwr = 3 THEN DevAvblStatus = 9
  • IF AdmitMaxPwr = 2 THEN DevAvblStatus = 8
  • IF AdmitMaxPwr = 1 THEN DevAvblStatus = 7
  • IF StartUpMode THEN DevAvblStatus = 1
  • IF CompOn THEN DevAvblStatus = 2
  • ELSE
  • DevAvblStatus := 0
