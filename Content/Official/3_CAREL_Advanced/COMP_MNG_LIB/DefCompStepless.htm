DefCompStepless

Provides all the necessary configuration data for Stepless compressor management

General operations

This function block provides all the necessary configuration data for stepless compressor management according to the compressor manufacturer and compressor model.

 

Inputs

Block pin list

Variable Data type UOM Description
CfgCompSteplessIn

CFG_STEPLESS_COMP

 - Stepless compressor configuration
CompManuf UINT -

Compressor manufacturer:

0: CUSTOM

1: BITZER

2: REFCOMP

3: FRASCOLD

4: HANBELL

5: FUSHENG

6: J&E HALL

7: COPELAND

8: DANFOSS
CompModel UINT -

"Compressor models supported"

En_DefCompParams BOOL - Enable this function block

 

Outputs

Block pin list

Variable Type UOM Description
Cfg_CompStepless CFG_STEPLESS_COMP - Stepless compressor configuration
En_CompStepless BOOL - Enable STEPLESS compressor
DefDone BOOL - Default values for the compressor are loaded
CurrManuf UINT - Current compressor manufacturer
CurrModel UINT - Current compressor model
