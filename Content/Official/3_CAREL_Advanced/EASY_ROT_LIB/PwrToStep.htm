General operations

"PwrToStep" is used with function blocks of "EASY_ROT_LIB" library only when the devices request is expressed in percentage. The maximum number of devices allowed is 32.

 

Conversion

The block returns a step request by a proportional conversion of the 0-100.0 REAL input power.

The conversion includes also the hysteresis: only when the power request pass a setpoint the step increments.
