Project description

This demo can be used to manage a single circuit of 6 devices or two circuit of 3 devices each one; the rotation follow the FIFO logic. The devices are compressors on-off.

 

Single circuit

The variable EnSingleRot = TRUE allows to use the single circuit, the request is a REAL 0-100.0, converted in step by Function Block "PwrToStep".

LoadUp and LoadDown timings are 10s.

 

Multi circuit

The variable EnSingleRot = FALSE allows to use the multi circuit rotation. It manages 2 circuits of 3 devices each one. The structure MngCircRot manages the circuits parameters and is parameterized by RotMng Function Block.

 

Devices availability

At the top of the RotationMng POU there is the control of availability of the compressors for Single and Multi rotation. This management can be enabled/disabled in the "Comp. cfg" menu.

 

Compressors management

In CompressorMng POU there is the management of the compressors; this was developed following the Demo_COMP_MNG_LIB and adapted for on-off compressors.

Go to "Comp. cfg" menu to set the compressor MinOnT, MinOffT and MinOnOnSameT timings.

 

Main default settings

Customize the DEV table in c.design with your own default values.
