MultiRot

Manages the FIFO rotation of on/off devices as a circuit.

General operations

"MultiRot" works with multiple circuits system, up to 32 on/off devices per circuit, following the FIFO logic.

The devices requests are reported by output array "ReqCirc_MR".

This block works as SingleRot, with a different output structure.

The request occurs by a BOOL array (which contains the circuit) of arrays (which contains the devices per circuits)

 

Feedback Input Output

The block provides feedbacks to "RotMng" by "MaxPwrCirc_MR", "MinPwrCirc_MR" and "ReqCirc_MR"; these feedbacks have to flow through all circuits.

For more information, see "Demo_EASY_ROT_LIB".

To set up a network of MultiRotation, please follow the guide in "RotMng" documentation, paragraph "How to implement more circuits".
