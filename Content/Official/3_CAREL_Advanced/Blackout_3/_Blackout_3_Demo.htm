Project description

This demo can be used to:

  1. See how to correctly manage the NVRAM/Blackout_3 blocks

 

Please carefully read the comments in c.strategy, in order to have correct management of the NVRAM:

  1. The NVRAM block management MUST stay before the Blackout_3 block
  2. Set the correct address of the NVRAM block, according to your application NVRAM management
