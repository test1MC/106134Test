CheckRetainMem

Monitors the number of writes to EEPROM memory.

General operations

The block shows as output the absolute number of writes to retain memory during the entire device life.

Every CheckT the block samples the number of writes to memory since last check. If this happens, MemWarning becomes true. If this number is greater than MaxWritingsNo for CountToTrig consecutive times, MemError is set to true.

 

Errors are automatically reset when a sample is lower than MaxWritingsNo or when the block is disabled.
