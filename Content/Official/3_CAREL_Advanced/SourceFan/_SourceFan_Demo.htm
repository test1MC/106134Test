Project description

This demo can be used to:

  1. Set all the SourceFan parameters, in order to understand the functioning of the Block
  2. Set the SourceFan as Chiller or HeatPump mode; both condensing/evaporation temperatures can be set by screen
  3. It contains complete unit of measure management for the User Interface
  4. It contains a simple fan animation.

Common functions are grouped into the same strategy page in order to allow better focus and to prepare that page for exporting.

 

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings

Step 1:

Set the main control parameters in "3. Fan cfg" menu.

Step 2:

Set the fan parameters.

Step 3:

Set the main parameters, enable the compressor and the SourceFan function block. It is also possible to set the unit of measure zone.

Step 4:

Set the temperature and control fan activation.

Step5:

Antifrost and Force request management can be set in "3. Fan cfg" menu.
