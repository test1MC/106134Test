SourceFan

Manages the source side ventilation on a "Chiller/Heat Pump" unit with up to 3 ON/OFF devices or a modulating fan.

The SourceFan function block allows the following functions:

  • Proportional control of condensing(CH)/evaporation(HP) temperature, managing the activation of maximum 3 steps or 1 modulating output
  • Speed-up every modulating fan start, and control of max-min request
  • Antifrost function to prevent the fans from freezing in cold climates
  • Prevent action in case of high condensing temperature (and thus high pressure)
  • Manual control of fan requests

 

FB operating conditions

The SourceFan function block starts whenthe following conditions are satisfied :

En_SourceFan = TRUE

 

General operations

The modulating fan (FanTyp=FALSE) is controlled by the analogue output, its continuous modulation is proportional across the entire control band.
The output values will range between "CfgFanReg.Inverter.LimMin" and "CfgFanDev.Inverter.LimMax", corresponding respectively to condensing temperature values equal to "CfgFanReg.Chiller.Setpoint" and "CfgFanReg.Chiller.Setpoint" + "CfgFanReg.Chiller.Differential". REAL output "InvReq" is the request for the modulating fan.
In this case, "Fan_1" output is used as activation for the device that controls the modulation of the ventilation unit.

 

The On/Off fans (FanTyp=TRUE) are controlled by digital outputs "Fan_1", "Fan_2", "Fan_3", which are activated and deactivated by control.
It is possible to set the differential for individual fans (CfgFanReg.OnOff.FanDiff) as a percentage of the total differential (CfgFanReg.Chiller.Differential), to increment control stability by overlapping the fan hysteresis.
"CfgFanReg.OnOff.FansDelay" parameter manages the activation/deactivation delay between fans.

 

If "CompOn = TRUE" then the condenser control digital outputs are enabled, while if "CompOn = FALSE" the condenser control digital outputs are disabled, and the devices can be activated by the high pressure prevention functions or by antifrost

The high pressure prevention function turns on the condenser control output bringing it to the maximum value to prevent the system safety devices from triggering (pressure switches, alarms based on transducers). It is necessary set the activation and hysteresis thresholds, respectively "CfgFanReg.Chiller.ForceSetpoint" and "CfgFanReg.Chiller.Differential".
The function is always enabled when compressors are active (CompOn = TRUE).

The antifrostfunction (present only with modulating fans) has the purpose of activatingventilation, even with the unit off, to prevent the fans from freezing.
Antifrost function is activated when the outside temperature falls below the set threshold (ExtTemp<SourceFanMng_AntiFrostCfg.Thrsh), and starts with a speed-up (SourceFanMng_AntiFrostCfg.StartUpSpeed) in order to start impeller rotation; this phase has a limited duration (SourceFanMng_AntiFrostCfg.StartUpT) and then ventilation is controlled with a low request (SourceFanMng_AntiFrostCfg.MinSpeed) able to keep the fans rotating and therefore avoiding freezing.
The antifrost turns off when the outside temperature exceeds the activation threshold by 2 ° C .
