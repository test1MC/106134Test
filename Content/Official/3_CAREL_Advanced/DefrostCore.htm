Defrost procedure

The defrost procedure is performed in 9 steps; if the defrost procedure is running, the current step is shown by the Status_DFC variable

 

Check defrost start condition (Status_DFC = 0)

The defrost is not running, the defrost procedure will start if the Run_DFC variable is set to TRUE for at least SyncDT_DFC seconds.

 

Wait for other circuits (Status_DFC = 1 and SyncTyp = 1)

If SyncTyp_DFC is set to 1 (separate defrost, only one circuit allowed to defrost at a time) the block will wait for the other circuits to end the defrost procedure

 

Decrease compressor speed (Status_DFC = 2)

The compressors need to be forced to the changeover capacity in order to reverse the cycle. The block will remain in this phase until ChgOverPwrReached_DFC is FALSE. Once ChgOverPwrReached_DFC becomes TRUE, the procedure moves to the next phase.

Timer for consecutive defrost starts

 

Wait before changeover (Status_DFC = 3)

The block will wait the time specified by DT_BeforeChgOver_DFC. At the end of the timer the procedure moves to the next phase.

 

Changeover (Status_DFC = 4)

Change over is performed (ReverseCycle_DFC = TRUE), fans will be forced off (FrcFanOff_DFC = TRUE) and after a 5 second delay the procedure moves to the next phase.

 

Defrost (Status_DFC = 5)

Compressors are forced to defrost capacity (FrcCompDfrPwr_DFC = TRUE).

This phase will last until Stop_DFC becomes TRUE and MinT_DFC has elapsed, or the MaxT_DFC has elapsed.

If a compressor alarm occurs in this phase, these two timers will restart.

 

Compressor slowdown/off and reverse cycle (Status_DFC = 6)
  • Compressors will slow down to minimum speed (FrcCompToMinPwr_DFC = TRUE), the reverse cycle is performed as soon as the DeltaP_DFC reaches the value of RevVlvDeltaP_DFC * 1.5 or after 60s from the start of this phase.

    • After reversing the cycle, the counter of DT_AfterChgOver_DFC starts, when this timer ends the procedure moves to the next phase.

     

    Dripping (Status_DFC = 7)

    Compressors are forced off (FrcCompOff_DFC = TRUE).

    This phase will last DripT_DFC seconds.

     

    Post-dripping (Status_DFC = 8)

    Fan are forced on (FrcFanOn_DFC = TRUE).

    This phase will last PostDripT_DFC seconds.

     

    Defrost synchronisation

    If there is more than one circuit, it is possible to synchronise defrost behaviour.

     

    Independent (SyncTyp_DFC = 0)

    Each circuit will start defrosting when it meets the conditions, no synchronisation is performed and the defrost can be running on more than one circuit.

     

    Separate (SyncTyp_DFC = 1)

    When a circuit requests to start defrost, if another circuit is performing a defrost, the block will wait until the other has finished the defrost procedure.

     

    Simultaneous (SyncTyp_DFC = 2)

    When a start request arrives, if the synchronisation timer (SyncDT) for the current circuit is running and another circuit starts the defrost (Run_DFC to TRUE), then the defrost procedure starts immediately, otherwise it will wait until the other has finished.

    If another circuit is running a defrost, the compressors in the circuit not running the defrost will be forced off, until the end of the defrost procedure.

     

    Inputs

    Block pin list

    Variable

    Data type

    		 UOM Description
    Start_DFC BOOL - Defrost start condition (request by other)
    Stop_DFC BOOL - Defrost end condition (request by other)
    ManStart_DFC BOOL - Defrost manual start command
    ChgOverPwrReached_DFC BOOL - ChangeOver capacity is reached
    CompsOn_DFC BOOL - Compressor On
    IntervalT_DFC UINT min Delay between 2 consecutive defrosts
    SyncDT_DFC UINT s Delay to check for simultaneous defrosts
    DT_BeforeChgOver_DFC UINT s Delay time before reversing the 4 way valve - defrost begins
    DT_AfterChgOver_DFC UINT s Delay time after reversing the 4 way valve - defrost ends
    MinT_DFC UINT min Minimum defrost duration
    MaxT_DFC UINT min Maximum defrost duration
    DripT_DFC UINT s Dripping duration
    PostDripT_DFC UINT s Post-dripping duration
    SyncTyp_DFC UINT - Defrost synchronisation type (0: Independent; 1: Separate; 2:Simultaneous)
    Sync_DfrRunOtherCirc_DFC BOOL - Another circuit is currently performing the defrost procedure (used to synchronise this defrost block with others for multi-circuit applications)
    DeltaP_DFC REAL bar Current delta pressure
    RevVlvDeltaP_DFC REAL bar Reverse valve min delta pressure to activate
    En_DFC BOOL - Enable the defrost function

     

    Outputs

    Block pin list

    Variable

    Data type

    		 UOM Description
    Run_DFC BOOL - Defrost is running
    Status_DFC USINT - Defrost status
    1. Not run
    2. Inhibitcircuit for synchronisation
    3. Decrease compressor speed
    4. Wait changeover time
    5. Reverse cycle and force fan off
    6. Defrost
    7. Decrease compressor speed and reverse cycle
    8. Dripping
    9. Post-dripping
    ReverseCycle_DFC BOOL - Reverse cycle
    FrcCompChgOverPwr_DFC BOOL - Go to changeover capacity
    FrcCompDfrPwr_DFC BOOL - Go to defrost capacity
    FrcCompToMinPwr_DFC BOOL - Force compressor to min capacity
    FrcCompOff_DFC BOOL - Force Off compressor
    FrcFanOff_DFC BOOL - Force Off source fan
    FrcFanOn_DFC BOOL - Force On source fan
