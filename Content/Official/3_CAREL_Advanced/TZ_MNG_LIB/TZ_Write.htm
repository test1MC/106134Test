General operations

TZ_Write acts as an intermediate function to the RTC_TZ_WRITE block by storing inside all the strings that can be passed to RTC_TZ_WRITE.

It also manages time zones that change based on the current year

Input pin "Set_TZW" allows to set the new time zone on the target.

"Set_TZW" works on the rising edge. See "Example 1" for more information
