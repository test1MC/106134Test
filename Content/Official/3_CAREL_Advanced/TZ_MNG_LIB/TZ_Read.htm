General operations

TZ_Read acts as an intermediate function to the RTC_TZ_READ block, codifying its output string to output pin "Zone_TZR".

Input pin "Read_TZR" allows to read the current time zone set on the target.

"Read_TZR" works on the rising edge. See "Example 1" for more information
