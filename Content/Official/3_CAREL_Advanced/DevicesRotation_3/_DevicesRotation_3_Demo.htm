Project description

This demo can be used to:

  1. Manage 2 circuits with 3 compressor steps, and 4 maximum devices per circuit
  2. Test main block functions (different types of rotation, capacity equalisation, unload sequence, compressor alarms…)

  3. Simulate a capacity request and retrieve from OS a system log that can be viewed in an Excel file

 

Notes

The Demo can be uploaded toa c.pCO and will work directly without any manipulation. The default configuration set is:

  • 2 circuit, 3 On/Off compressors
  • FIFO, Equalised, CCpppppp
  • LdUpT, LdDwnT and DTUnld=10s

The demo uses the compressor library. To test device rotation with a compressor with inverter, you can easily drag&drop a Comp_STEPLESS block from the block library.

Is possible to speed up the test of the hour counters by setting the “Adjust” variable to a value other than 0 (this is found inside the local variables of POU HoursCounters).

The Demo has a log set so as to log the simulation and export it to view in an Excel file and see it graphically. The variables logged are PwrReq, DevRot array and ModulPwrReq.
