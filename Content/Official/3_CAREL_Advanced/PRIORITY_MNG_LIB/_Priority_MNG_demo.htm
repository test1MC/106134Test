Project description
Features

This demo can be used to test the communication between two c.pCO by sharing and exchanging values of some variables. The Demo provides different types of communication in order to test:

  1. ReadPoll/Write change communication;
  2. NoRead/Write change;
  3. ReadPoll /noWrite.
Proper operation

If the system is working in right way:

  1. No offline alarm are present
  2. The first mask reports the title, indicating the type of communication (Read/NoWrite, Read/Write and NoRead/Write) and 6 variables to permit values changing
  3. Changing the variables values in the Master Read/Write page will change the value also in the Slave Read/Write page and vice-versa.
  4. Changing the variables values in the Master NoRead/Write page will change the value in the Slave Read/noWrite page.

  5. Changing the variables values in the Master Read/NoWrite page will produce no changes in slave Slave pages.

Main settings
Step 1

Install the Demo_Priority_MNG on the first c.pCO, using the Master protocol

Step 2

Install the Demo_Priority_MNG on the second c.pCO, using the Slave protocol
