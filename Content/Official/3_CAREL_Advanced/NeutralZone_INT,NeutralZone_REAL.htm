General operations

These blocks are used to calculate capacity request based on the neutral zone method, which is able to find the capacity value that balances the “cooling request” or the “heating request”. 

When Input value is in the neutral zone, the Output doesn’t change. When the Input exits the zone, Output is increased or decreased (depending on the band and the configuration, as can be seen in the graph). In this way, the block can find the middle value after a certain amount of time.

 

Force off threshold

FrcOffThrsh input is used to force off the unit when Input is below the threshold and avoid it running with very low capacity requests. To not use this function, , simply set a very low or very high value that will never be reached.

 

Function settings

There are some additional parameters, DiffRegIncr, DiffRegDecr, MinT0, MaxT0, MinT100 and MaxT100. These parameters are used to adjust the system reaction to an Input variation when it is outside the neutral zone. The higher the time and smaller the differentials, the faster the system reacts. Try to find a good compromise between speed and stability, otherwise there will be fast control that never reaches balance or control that is too slow . See below to understand how these parameters works.

 

Manual Mode

When ManMode is true, the algorithm is bypassed and Output takes exactly the same value as ManOut. This allows control to be interrupted and capacity to be forced to any value. When ManMode is reset to false, normal control restarts.

Variable

Data type

 UOM Description
Input_NZ REAL - Input value from probe

DiffRegDecr_NZ REAL - Differential for the decreasing band
MinT0_NZ TIME - Minimum time to reach minimum capacity
MaxT0_NZ TIME - Maximum time to reach minimum capacity
MinT100_NZ TIME - Minimum time to reach maximum capacity
MaxT100_NZ TIME - Maximum time to reach maximum capacity
FrcOffThrsh_NZ REAL - Set point to force capacityrequest to 0
Reverse_NZ BOOL - Reverse the function (false=cooling, true=heating)
ManMode_NZ BOOL - Enable manual mode
ManOut_NZ REAL - Specify which value Output must take when in manual mode
En_NZ BOOL - Enable the block
