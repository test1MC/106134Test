Project description

This demo can be used to:

  1. Manage 2 circuits with 3 step compressors, and maximum 4 devices per circuit
  2. Test main block functions (different types of rotations, capacity equalisation, unload sequence, compressor alarms…)

  3. Simulate a capacity request and retrieve a log of the system from the OS to view as an Excel file

 

Notes

The Demo can be uploaded to a c.pCO and will work directly without any manipulation. The default configuration set is:

  • 2 circuit, 3 On/Off compressors
  • FIFO, Equalised, CCpppppp
  • LdUpT, LdDwnT and DTUnld=10s

The demo uses the compressor library. To test device rotation on a compressor with inverter, simply drag&drop a Comp_STEPLESS block from block library.

To speed up the hour counter test, set the “Adjust” variable to a value other than 0 (this is found inside the local variables of POU HoursCounters).

The Demo has a log set that can be used to log the simulation for exporting and viewing in an Excel file graphically. Variables logged are PwrReq, DevRot array and ModulPwrReq.
