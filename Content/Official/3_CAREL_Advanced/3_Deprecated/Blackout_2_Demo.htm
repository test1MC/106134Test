Project description

With this demo it is possible to:

  1. See how to manage correctly the NVRAM/Blackout_2 blocks

 

Please read carefully the comments in c.strategy, in order to have a correct management of the NVRAM:

  1. The NVRAM block management MUST stay before the Blackout_2 block
  2. Set the correct address of the NVRAM block, according to your application NVRAM management
