Variable Data type UOM Description
Speed_StartUpSpeed REAL - Start up speed

Speed_MaxGoUpSpeedRate REAL rps/s Max. increase speed rate (going to target speed in defrost)

0=null
1=SSH

3= DLT

CstmEnv_SuctPMin REAL barg Custom envelope low suction pressure limit

CstmEnv_DscgTempLimDistance REAL - Action distance from high temperature limit (to reduce speed rate)
CstmEnv_DscgTempPause INT - Pause between speed reductions when discharge temp. is over control limit
CstmEnv_DscgTempSpeedReduction REAL - Speed reduction percentage when discharge temp is over control limit
CstmEnv_OutEnvMaxTSpecif INT - Out of envelope alarm delay time to be used when specified by VAR_INPUT
CstmEnv_DscgTempDTActionPauseRes REAL - Value of discharge temp. increase that requires a new action
CstmEnv_MaxspeedXrps REAL rps

Dynamic maximum speed limit (rps). This variable is used to reduce the normal limit Speed_MaxSpeedRpsCustom. If this variable is different than 0.0 then the normal Speed_MaxSpeedRpsCustom is bypassed by CstmEnv_MaxSpeedXrps (Default value is 0.0rps).

This variable is a retain flag. Remove the retain flag for frequent writing to preserve retain memory.
CstmEnv_MinspeedXrps REAL rps

Dynamic minimum speed limit (rps). This variable is used to reduce the normal limit Speed_MaxSpeedRpsCustom. If this variable is different than 0.0 then the normal Speed_MaxSpeedRpsCustom is bypassed by CstmEnv_MaxSpeedXrps (Default value is 0.0rps).

This variable is a retain flag. Remove the retain flag for frequent writing to preserve retain memory.
PmpDwn_En BOOL - Enable pump down procedure
PmpDwn_SuctPThrsh REAL - Suction pressure threshold for pump down
PmpDwn_MaxT INT - Maximum pump down time
