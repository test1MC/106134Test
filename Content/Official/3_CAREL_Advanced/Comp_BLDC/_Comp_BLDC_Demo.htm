Project description

This demo can be used to:

  1. Manage the operation of the compressor, piloting the Evd -Evo driver and the Power + inverter with the Modbus protocol.
  2. See examples of using alarms
  3. Start a project to implement with other functions

Common functions are grouped on the same strategy page in order to allow better focus and to prepare that page for exporting.

Main default settings

The following steps are necessary in order to use the demo:

 

Main settings

Step 1:

Select the correct models of compressor to be used. This parameter is available in the main menu under item Power +

Step 2:

Write the default parameters.

Step 3:

Switch on the unit.

Step 4:

Check the input probe values: the working point must be inside the envelope (refer to the Envelope document). This parameter is available in the main menu under item Comp. BLDC

Step5:

Set required capacity from the main screen.

The BLDC control analyses the variation in pressure when the Power + is running. If the pressure values do not vary within 15 seconds (default), the cycle is stopped. This condition causes a "Start up failure controls" alarm. The system attempts to restart 5 times (default), after which it stops.

If Demo_Comp_BLDC is uploaded to the simulator, when the Power + is running, vary the pressures connected to the simulator to avoid it stopping.

The BLDC control analyses the differential between the suction and discharge pressure. If the pressure difference exceeds the set point, the system starts the equalisation function, and displays the message "Max PD prevent start".

If Demo_Comp_BLDC is loaded on the simulator, at the request to run the PowerPlus, vary the discharge pressure and bring it closer to the suction pressure. Afterward, when the PowerPlus is running, restore the discharge pressure to the value required for the envelope.
