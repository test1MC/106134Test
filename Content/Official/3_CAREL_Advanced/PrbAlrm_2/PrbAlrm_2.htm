General operations

AlrmPrb_2 controls that the input value is within the limits imposed by the pin and manages the probe fault alarm. It can also recognise probe disconnection.

Min and max values set to the pin must be equal to the values used in I/O Editor.

In order to detect probe disconnection, set ActPrbAlrmDelta to a value different from zero

For active probes, it is possible to manage a lower threshold in order to activate an alarm before reaching the minimum value, as shown in the image.

Two types of delay time are managed:

  • ActPrbAlrmDT delays the effect of the input value in the lower threshold (only for active probes)
  • AlrmDT delays the alarm after the condition has occurred
