Project description

This demo can be used to:

  1. Monitor a probe error and detect its disconnection
  2. Test the behaviour of a probe
Main default settings

Default settings are:

Input channel U1
Probe type 0..20 mA
Min 0.0
Max 20.0
AlrmDT 0 sec
ActPrbAlrmDT 2 sec

The following steps are necessary in order to use the demo:

Main settings
Enable FB PrbAlrmMng.En_PrbAlrm = TRUE
