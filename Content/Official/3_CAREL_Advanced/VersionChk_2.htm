General operations

The versionChk_2 function block checks the version of the application. If this has been changed, at pCO startup, the "VerMismatch" output variable is set to TRUE.

 

Structures

Description of structures defined and used by the F/FB

Structure name Variable Data type Description
VERSIONCHK_APPVER   Version of the application
  X UINT X version of the application
  Y UINT Y version of the application
  Z UINT Z version of the application
  D UINT D version of the application
  IsBeta BOOL Enable Beta version
  IsDemo BOOL Enable Demo version

 

Inputs

Block pin list

Variable Data type UOM Description
Version VERSIONCHK_APPVER - Version of the application

 

Outputs

Block pin list

Variable Data type UOM Description
VerMismatch BOOL - Check the application version every startup and inform the user if the version has been changed
