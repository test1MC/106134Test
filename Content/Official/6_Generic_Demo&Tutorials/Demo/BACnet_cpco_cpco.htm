Project description

This demo shows how to define the communication settings, to provide in a c.pCO application, for having a Client-Server connection via BACnet TCP/IP and MS/TP.

The demo application shares the same c.strategy (variable declaration only) and c.mask part for both the c.pCO, Client and Server; c.design, on the other hand, contains two different configurations, one for the Client and one for the Server c.pCO. Once the device instance and the operating mode has been entered in the "Client or Server"network, the client or server configuration screen will be shown.

When using the MS/TP protocol, the station address setting is necessary.

The application can read/write 3 variables, one binary, one analogue and one integer.
