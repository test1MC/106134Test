Project description

This demo shows how the IMPORT_PARAMS and EXPORT_PARAMS c.function block works: it allows you to set the value of three variables (BOOL, INT and REAL) by screen interface, and export or import their values via a text file called "EXPORT_xx.txt" saved to USB or internal flash memory.

A screen is available that manages the settings of the variables, plus a screen that manages parameter export and import.

 

 

The Import-Export of parameters only works with DEV variables that have a user value.
