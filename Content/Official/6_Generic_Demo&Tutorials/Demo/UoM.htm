Demo/Test application toperform conversions between different units of measure

 

Project aims

The project manages conversions between the following units of measure:

  • International System [T(°C), P(kPa), …]
  • Imperial USA [T(°F), P(psi), …]
  • Imperial UK [T(°F), P(psi), …]
  • Canadian [T(°F), P(psi), …]
  • Carel LON card
  • International System (Bar)

 

Project description

The project can be used to change the units of measure relating to the values of:

  • Ambient temperature, from an NTC probe connected to U1
  • Ambient set point, set by screen
  • Differential, set by screen
  • Suction pressure, set by screen

UoM conversion zone can be changed by the "STD_UOM_ZONE_UI" variable.

 

The "STD_UOM_ZONE_UI" variable sets the conversion zone for the User Interface: this is a Retain variable, but needs to be initialised every time c.pco starts.
The initialization code has to be set in the Timed Loop of ST code in c.mask.

        IF FirstCycleDone = FALSE THEN
          FirstCycleDone := TRUE;
          SetUoMZone (STD_UOM_ZONE_UI);
        END_IF;

See also UoM configuration from UoM_presentation.ppt

See also UoM presentation available in KSA
