Project description

This demo shows how the EXPORT_ALARMS c.function block works.

This demo contains the alarm export management, also present in the latest version of the project template, and the standard alarm management part that manages the alarm logs.

This demo has 10 defined alarms, whose triggers are connected to the DIN1 - DIN10 ona c.pCO.

This demo contains a mask that manages the export and reset of the alarm log.
