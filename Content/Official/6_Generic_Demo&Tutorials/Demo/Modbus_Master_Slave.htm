Project description

This demo shows how to implement a Modbus line managing two different protocols, one for the Master side and one for the Slave side. The c.pCO slave configuration manages a simple strategy logic that inverts the value of a Boolean variable, sent acrossthe Modbus network; this is usedto understand whether or not communication is active.
