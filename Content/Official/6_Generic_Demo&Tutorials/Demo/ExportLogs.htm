Project description

This demo can be used to manage the export logs from the application using the EXPORT LOGS function block. The variable that is logged is a REAL variable connected to Universal Channel 1 set as CAREL NTC

This demo contains:

  • c.strategy POU that manages EXPORT LOGS
  • c.mask worksheet with all the functions related to the EXPORT LOGS
  • c.design log configuration that contains two logs

 

The screens included in the project are shown below

Screendescriptions

The value of "Select log to export" matches the position of the log in the log editor, starting from 0
 

This screen is enabled only if the export interval is set as INTERVAL

 

If there are multiplelogs, remember to update the LogIdx class in order to have consistent names, or setit to -1 to always export all logs
