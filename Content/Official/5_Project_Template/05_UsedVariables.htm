List of variables used

Following is a list of variables used by Project Template:

  • Application versions
  • Counters
  • Parameter Import/Export
  • Alarm log export
  • Blackout
  • Daylight savings time
  • Quick menu
  • Retain memory
  • Unit On/Off
  • Application start jump
  • Miscellaneous management
  • Time zone management
  • General application and hardware information
  • Change language management
  • General menu management
  • Main menu management
  • Settings menu management
  • Password management
  • Alarm management
  • Unit of measure management

 

for the variable limits, see Data types.
