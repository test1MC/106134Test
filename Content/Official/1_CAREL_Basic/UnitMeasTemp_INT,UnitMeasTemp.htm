General operations

If needing to convert a temperature differential, use UnitMeasTempDelta_ instead.

 

The same variables, the one used in the algorithm (base) and the one for the interface (display), must both be connected to input and output.

 

BaseUnit and DisplayUnit inputs are used to specify the unit of measure used for that value.

When an input changes, the other is updated accordingly. If both change, base has priorityand display is updated.

At start-up, DisplayTemp is immediately updated based on BaseTemp.

The block is able to recognise a change in thedisplay unit at runtime and respondimmediately with the correctconversion.
