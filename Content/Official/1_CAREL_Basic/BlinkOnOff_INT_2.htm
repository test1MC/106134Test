BlinkOnOff_INT_2

Generate a cyclical programmed time sequence of impulses.

The block is not present in CAREL_Basic library but, it is downloadable from KSA website.

General operations

If In is true, Out is set cyclically to TRUE and FALSE, Out stays TRUE for OnT seconds, then is set to FALSE for OffT seconds, Out will start always with TRUE

If In is false, Out is always FALSE.

Changing OnT or OffT on fly is allowed.
