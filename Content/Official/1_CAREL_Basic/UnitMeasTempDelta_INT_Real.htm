General operations

These functions are used to convert a differential in Celsius to a differential in Fahrenheit.

 

If needing to convert a temperature value (not differential), use CelToFar_ instead

 

The same variables, the one used in the algorithm (base) and the one for the interface (display), must both be connected toinput and tooutput

 

BaseUnit and DisplayUnitinputs are used to specify the unit of measure used for that value.

When an input changes, the other is updated accordingly. If both change, base has priorityand display is updated.

At start-up, DisplayTempis immediately updated based onBaseTemp.

The block is able to recognise a change in thedisplay unit at runtime and respondimmediately with the correctconversion.
